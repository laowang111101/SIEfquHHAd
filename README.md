# 手机商城管理系统 java717

## 项目概述

手机商城管理系统是一款基于SpringBoot与Vue的前后端分离架构的系统。它集成了商品信息管理、商品评价管理、商品分类管理、公告管理、订单管理等多种功能模块，适用于现代电商业务需求。

## 技术栈

- 后端：SpringBoot
- 前端：Vue
- 数据库：MySQL
- 开发工具：IntelliJ IDEA, VSCode
- 插件与库：MyBatis-Plus, Element UI

## 功能模块

### 系统角色

- 用户
- 超级管理员

### 功能介绍

- **商品信息管理**：实现商品的增删改查等操作。
- **商品评价管理**：管理用户对商品的评价，维护商城口碑。
- **商品分类管理**：对商品进行合理分类，便于用户浏览。
- **公告管理**：发布和管理商城公告。
- **订单管理**：处理用户订单，跟踪订单状态。
- **用户管理**：管理商城用户，维护用户数据。
- **个人中心**：用户可查看个人信息，修改资料等。
- **个人后台**：管理员操作后台，进行系统管理与设置。

## 运行环境

- 操作系统：Linux/Windows
- 数据库：MySQL 5.7+
- Java版本：JDK 1.8+
- Node版本：Node.js 10.0+
